리눅스 터미널 명령어

  • 파일 및 디렉터리 관련 명령어
    • ls [l, a, f]: 디렉토리 내용을 나열합니다.
      • -l : 파일의 상세 정보를 출력
      • -a : 숨긴파일 모두 출력
      • -f : 파일 종류를 구분해서 표시 해준다.
    • cd : 디렉터리를 변경합니다
    • pwd : 현재 작업 중인 디렉터리의 위치를 출력
    • touch : 새로운 빈 파일 생성
    • mkdir [p, m] : 현재 위치에 새 디렉터리를 생성
      • -p : p 뒤에 dir1/dir2/dir3을 붙이면 현재 디렉터리에서 부모 디렉터리를 포함해 그 안에 dir2을 만든다 그 안에 dir3을 만들고
      • -m : 디렉터리의 권한 설정임 mkdir -m 700입력하면 본인 (읽,쓰,실)제외 모든 이들은 (읽,쓰,실) 불가
    • rmdir [p] : 디렉터리 파일을 삭제할 때 사용함
      • -p : 디렉터리
    • cp [r, i, v] : 파일이나 디렉터리
      • -r : 복사할 디렉터리 안의 내용물을 포함해서 복사
      • -i : 옮길건지 일일히 물어본다 (y,n으로 대답 가능)
    • mv [i, f] : 파일이나 디렉터리의 이름을 변경하거나 위치
      • -i : 이미 존재하는 경우, 덮을거냐고 물어보는거
      • -f : 있으면 강제로 덮어쓰기함
    • rm : 파일이나 디렉터리를 삭제합니다.
  • 시스템 정보 관련 명령어

    uname : 운영체제 정보 출력

    hostname : 호스트 이름을 출력, 변경

    date : 현재 시간과 날짜 정보 출력

  • 프로세스 및 작업 관리

    ps : 현재 실행중인 프로세스 상태를 보여줌

    top : 시스템의 전반적 상태와 실행중인 프로세서 리스트를 보여줌.

  • 검색 및 정규 표현식 관계

    grep : 정규 표현식과 일치하는 문자열 검색

    find : 파일 검색

  • 권한 및 소유자 변경

    chmod,chown 등
