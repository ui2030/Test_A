인공지능

설명하는 문제 = 신경망의 응용절차, 경사하강법, 델타 규칙

푸는 문제 = 호필드 네트워크

  • 신경망의 응용절차
    • 인공 지능 - 학습, 추론 등을 수행하는 사람의 뇌를 컴퓨팅 기술로 모델화 하는 것.
    • 기계 학습 - 알고리즘을 이용해 데이터를 분석하고 분석을 이용해 학습한다. 학습으로 얻어진 정보를 기반으로 판단이나 예측 한다. 주로 학습에는 지도,비지도,강화,준지도 학습이 있고, 활성화 함수에는 항등, ReLU, sigmoid 함수가 있다. 주로 실생활에선 신용카드 사기 탐지 등에 사용된다.
    • 신경 회로망 - 인공 뉴런들이 연결 강도들에 의해 연결되어 있고, 이 연결 강도들은 학습 과정에서 최적화되어 패턴 인식, 분류, 예측 등의 작업을 수행하는 네트워크들이다. 이미지 인식(OCR)이나, 음성 인식 같은 복잡한 패턴을 인식하는데 사용한다.
    • 딥러닝 - 심층신경회로망을 효과적으로 학습해서 패턴 인식에 활용한다. 일반적으로 은닉층이 두개 이상인것을 심층신경회로망이라 부르는데, 잘 사용하지 않았다. 이유는 그래디언트 소실, 과적합 떄문인데 2000년대 들어서 개선(CNN, DNN등)이 되어 사용되었다. 주로 구글의 알파고나 자율주행 자동차의 사물인식, 음성인식 시스템에 사용되었다.

  • 경사하강법

    딥러닝 알고리즘을 학습시킬 때 사용되는 최적화 방법 중 하나로 학습을 통해 가중치 w를 더 좋은 값으로 바꾸는 방법 원리 : 현재 가중치 w와 현재 오차 E(w)가 존재하면, 기울기가 0이되는 곳이 최적해라는 것을 이용하여 오차를 감소시키는 가중치를 조정해서 최적해를 찾는 방법임.

공식 : w' = w - η∇E 문제점 : 경사 하강법은 일정 영역 내에서 모델의 기울기를 점점 줄여나가며 기울기가 0이 되는 지점을 찾는 최적화 방법이다. 그러나 일정 영역 내에서의 모델의 기울기를 찾는 것이기 때문에, 기울기가 0이 되는 지점을 찾았다 하더라도 해당 지점이 국소적인 영역, 즉 지역 최적값인지, 혹은 모델 전체, 즉 전역 최적값인지 알 수 없다. 정리하자면 경사 하강법을 통해 가장 좋은 답을 찾더라도 그것이 모델 전체에서 가장 좋은 답인지는 알 수 없다

  • 델타 규칙

    학습 데이터로 훈련하고, 구한 값이 정답에 가까워지기 위해 입력값에 영향을 주는 가중치를 갱신하면서 점점 정답과 근접해간다. 학습률이 너무 커도, 작지않게 가중치를 조절한다.

    w <- w + αex 여기서 α는 ( 0 < α ≤ 1) α는 학습률에 해당한다.

    이렇게 αex와 w(가중치)를 더해 새로운 가중치를 만들어 낸다.

    새로운 가중치를 Δw라고 가정하면 수식으로는 w ← w + Δw 이다.

역전파 알고리즘에서 문제는 층 수가 많을수록 출력층의 오차가 입력 층까지 제대로 전파되기 힘들다는 점이었다. 해당 문제의 해결책으로 CNN, 즉 합성곱 신경망이 제기되었다.

CNN은 합성곱신경회로망 필터의 계수와 에러를 미니마이징으로 조정이 가능하다 그러니까 잘 될 수 밖에 없다 영상 인식에 주로 쓰인다.
